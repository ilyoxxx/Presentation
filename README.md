## `$ ls projects/ --featured`

<div align="center">

### ✒️ SignPad — Votre téléphone devient le stylo

*Signature électronique 100% locale, sans serveur, sans compte.*

</div>

<table>
<tr>
<td width="60%">

**Le concept :** générez un QR code depuis votre ordinateur, scannez-le avec votre téléphone, signez du doigt en plein écran. La signature arrive instantanément sur l'ordinateur — **en clair, sur le disque, jamais sur un serveur distant**.

Aucune app à installer sur le téléphone, aucun compte à créer, aucun octet n'atteint Internet : tout transite sur le réseau local.

**Fonctionnement en 4 gestes :**

| Étape | Action |
|:---:|:---|
| `01` | **Générer** — un QR code unique est créé pour la session |
| `02` | **Scanner** — avec l'appareil photo, sans rien installer |
| `03` | **Signer** — du doigt, en plein écran, en mode paysage |
| `04` | **Recevoir** — la signature arrive aussitôt et se range dans l'historique |

</td>
<td width="40%">
